# Portafolio — Bryan A. Ramírez

Next.js, TypeScript y Tailwind CSS.

## Desarrollo local

```bash
cd bryan-portafolio
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Hoja de vida (PDF)

El archivo `public/cv_bryan.pdf` es el currículum descargable. La página `/cv` incluye un botón para abrirlo.

## Despliegue (Vercel)

Sube el repositorio a GitHub e importa el proyecto en [vercel.com](https://vercel.com).
